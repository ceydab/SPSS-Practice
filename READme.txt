# SPSS-Projects
Welcome to the SPSS Projects repository! This repository contains two statistical reports in .pdf utilizing SPSS visualization.

## **Projects Included**
1. **Hypothesis Test on Player Stats**: This report indicates the hypothesis of a player scoring a certain amount under the given circumstances.
2. **Data Plotting**: This report indicates visualization of player data from a given year.

Each project is contained within its own .pdf file, along with outputs and explanations of the SPSS visualization.

## **Note**
These reports were written as a part of my bachelor's level statistics course.
